[PATCH] net: ethernet: mediatek: mtk_eth_soc: Add of_node_put() before goto

[Date Prev][Date Next][Thread Prev][Thread Next][Date Index][Thread Index]

 

  • Subject: [PATCH] net: ethernet: mediatek: mtk_eth_soc: Add of_node_put() before goto
  • From: Nishka Dasgupta <nishkadg.linux@xxxxxxxxx>
  • Date: Tue, 16 Jul 2019 11:25:04 +0530
Each iteration of for_each_child_of_node puts the previous node, but in
the case of a goto from the middle of the loop, there is no put, thus
causing a memory leak. Hence add an of_node_put before the goto.
Issue found with Coccinelle.

Signed-off-by: Nishka Dasgupta <nishkadg.linux@xxxxxxxxx>
---

 		err = mtk_add_mac(eth, mac_np);

+			of_node_put(mac_np);
