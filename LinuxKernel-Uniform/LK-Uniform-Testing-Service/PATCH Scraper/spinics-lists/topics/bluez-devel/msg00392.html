Hello everyone,
I've sent this patch before, but for some reason I didn't got either a positive or even negative response.
Problem is like this, when we're doing a periodic scan we should be able to do bonding, but according to the code we can't, so then something is wrong, either we can't do periodic scanning, or we need to fix a bug. 

This patch tries to fix this bug, and applies against 3.35.

Thanks,
Manuel Naranjo
Wireless Cables Inc

--- hcid/adapter.c.orig	2008-06-28 19:59:50.000000000 -0300
+++ hcid/adapter.c	2008-06-28 20:05:47.000000000 -0300
@@ -2792,7 +2792,7 @@
 	str2ba(address, &bdaddr);
 
 	/* check if there is a pending discover: requested by D-Bus/non clients */
-	if (adapter->discov_active || (adapter->pdiscov_active && !adapter->pinq_idle))
+	if (adapter->discov_active)
 		return in_progress(msg, "Discover in progress");
 
 	pending_remote_name_cancel(adapter);
