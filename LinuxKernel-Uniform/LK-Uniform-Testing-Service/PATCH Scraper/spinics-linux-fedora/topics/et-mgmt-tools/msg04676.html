Hi,

Virtual Machine Console can not send
"Ctrl+Alt+Delete" to GuestOS from menu [Send key]-[Ctrl+Alt+Delete].
and
"Ctrl+Alt+Backspace" to GuestOS from menu [Send key]-[Ctrl+Alt+Backspace].

This patch fix this issue.

Signed-off-by: Tomohiro Takahashi <takatom@xxxxxxxxxxxxxx>

Thanks,
Tomohiro Takahashi

diff -r e03490a72a2e src/vmm-details.glade
--- a/src/vmm-details.glade	Mon Feb 09 14:55:18 2009 -0500
+++ b/src/vmm-details.glade	Fri Feb 13 11:42:07 2009 +0900
@@ -217,15 +217,15 @@
                 <child>
                   <widget class="GtkMenu" id="details-menu-send-key_menu">
                     <child>
-                      <widget class="GtkMenuItem" id="details-menu-send-cad">
+                      <widget class="GtkMenuItem" id="details-menu-send-cab">
                         <property name="visible">True</property>
                         <property name="label">Ctrl+Alt+Backspace</property>
                         <property name="use_underline">True</property>
-                        <signal name="activate" handler="on_details_menu_send_cad_activate"/>
-                      </widget>
-                    </child>
-                    <child>
-                      <widget class="GtkMenuItem" id="details-menu-send-cad1">
+                        <signal name="activate" handler="on_details_menu_send_cab_activate"/>
+                      </widget>
+                    </child>
+                    <child>
+                      <widget class="GtkMenuItem" id="details-menu-send-cad">
                         <property name="visible">True</property>
                         <property name="label">Ctrl+Alt+Delete</property>
                         <property name="use_underline">True</property>
