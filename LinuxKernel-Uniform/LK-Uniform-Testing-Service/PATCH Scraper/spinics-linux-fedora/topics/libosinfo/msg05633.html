The <kernel-url-argument/> attribute is what has to be passed to kernel
whenever performing a tree based installation.

For rhel < 7, "method" should be used. For rhel >= 7 "inst.repo" should
be used instead.

https://gitlab.com/libosinfo/osinfo-db/issues/28
