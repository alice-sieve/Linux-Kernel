This patch series, together with some small bug fixes/clean-ups, tries
to improve target, script, and avatar disks in a way that apps do not
actually have to set them up.

The changes do not break the current state as, if set, those disks are
going to be used.

With those changes, a lot of logic can be removed from the apps dealing
with unattended installations (mainly from GNOME Boxes).
