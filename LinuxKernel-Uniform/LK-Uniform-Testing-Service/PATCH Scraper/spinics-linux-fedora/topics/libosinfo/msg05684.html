isodata files are not part of libosinfo anymore. Nowadays they're part
of osinfo-db/test/isodata.

It's been done ir order to avoid testing osinfo-db together wih
libosinfo tests. Now we have all the tests done (via make check) as
part of osinfo-db itself.

I've adapted the patch for osinfo-db. So, no need to submit a v2.
