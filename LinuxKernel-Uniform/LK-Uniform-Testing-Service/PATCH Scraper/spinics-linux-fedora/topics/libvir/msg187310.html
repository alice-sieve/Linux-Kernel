'bochs'. Add implementation and tests for qemu.



Usually we split the XML addition from the driver impl,
that is:
* conf: schema, documentation, XML parsing and formating and XML test
 case
* qemu: qemu_command.c changes, xml2argvtest

Given that the enum checks would require the conf: patch to touch qemu
anyway and the changes are minimal, I will not ask you to split them,
but at least s/conf/qemu/ in the commit summary.
