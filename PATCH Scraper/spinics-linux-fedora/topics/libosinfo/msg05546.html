Before version 7, Debian releases where versioned using "X.0"
instead of simply "X", and before version 6.0 the full name also
included "GNU/Linux"; in any case, the version number (rather
than the codename) has always been the primary way releases are
labeled. Update <name>s accordingly.

Source: https://www.debian.org/releases/

Signed-off-by: Andrea Bolognani <abologna@xxxxxxxxxx>
