Enhance help -D to parse NT_VMCOREDD ELF notes present in KDUMP vmcore.

Example:
crash> help -D
[...]
num_vmcoredd_notes: 2
            notes[0]: 7ff9a578d010 (NT_VMCOREDD)
                      name: "cxgb4_0000:02:00.4"
                      size: 33558464
            notes[1]: 7ff9a778e010 (NT_VMCOREDD)
                      name: "cxgb4_0000:03:00.4"
                      size: 33558464
[...]

Signed-off-by: Rahul Lakkireddy <rahul.lakkireddy@xxxxxxxxxxx>
