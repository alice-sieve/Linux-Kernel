kernel commit 7dc603c902 ("sched/fair: Fix PELT integrity for new
tasks") introduced TASK_NEW state. This patch adds support for it
as 'NE' state.

Signed-off-by: Kazuhito Hagio <k-hagio@xxxxxxxxxxxxx>
